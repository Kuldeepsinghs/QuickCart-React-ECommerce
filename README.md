# 🛒 QuickCart – E-Commerce Web Application

QuickCart is a responsive e-commerce web application built using React.js. It allows users to browse products, view details, add items to cart, and manage wishlist with a clean and user-friendly interface.

---
https://quick-cart-react-e-commerce.vercel.app

## 🚀 Features

- 🔐 **User Authentication**
  - Login, Register, Logout using Firebase
  - Email verification support

- 🛍️ **Product Listing**
  - Fetch products dynamically using REST API
  - View detailed product information

- 🔎 **Search Functionality**
  - Real-time product search

- 📂 **Category Filtering**
  - Filter products by Men, Women, Jewellery, Electronics

- 🛒 **Cart System**
  - Add/remove items
  - Increase/decrease quantity
  - Dynamic total price calculation
  - Data persistence using localStorage

- ❤️ **Wishlist Feature**
  - Add/remove products using heart icon
  - Toggle wishlist (❤️ / 🤍)
  - Wishlist count shown in navbar
  - Persistent using localStorage

- 🔒 **Access Control**
  - Users must login to add items to cart or wishlist

- 🔄 **Session Handling**
  - Cart is cleared on logout to avoid data mixing

---

## 🛠️ Tech Stack

- **Frontend:** React.js, JavaScript, HTML5, Tailwind CSS  
- **Routing:** React Router  
- **State Management:** Context API  
- **Backend Services:** Firebase Authentication  
- **API:** FakeStore API  
- **Storage:** localStorage
